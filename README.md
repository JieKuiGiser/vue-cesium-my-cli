# 基于vue-cesium-vite 完善的cesium脚手架

## Build Setup

```bash
# clone the project
git clone

# enter the project directory
cd dir

# install dependency
npm install

# develop
npm run dev
```

## Build

```bash

# build for production environment
npm run build
```
